# **App Name**: ThriftersFind OMS

## Core Features:

- Admin Authentication: Secure login to protect dashboard access.
- Dashboard Overview: Display summarized metrics for total customers, orders, and sales.
- Customer Management: View, add, and edit customer profiles with contact information and order history.
- Order Management: Create, update, and manage customer orders; include fields for order details, payment status, payment mode and delivery info.
- Smart Order Status Updates: AI tool provides admins suggested updates to an order's status based on customer data and historical patterns.
- Reporting & Analytics: Generate reports on total sales, top customers, and courier usage, with options to export or print;.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) to ensure a clean, neutral backdrop.
- Accent color: Vibrant magenta (#FF00FF) for highlighting key actions and information.
- Body and headline font: 'Inter' sans-serif for a clean and modern user interface.
- Simple and professional icons for easy navigation. Consider icons to represent customers, orders, and reports clearly.
- A clean and intuitive layout with a clear hierarchy to facilitate admin tasks.
- Subtle transitions for loading data and state changes. For example, use fade-in effects for newly loaded content and gentle slide-in animations for sidebars.
